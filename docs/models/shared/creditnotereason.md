# CreditNoteReason

A reason for this credit note, which can be one of "Duplicate", "Fraudulent", "Order change", "Product unsatisfactory"


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `Duplicate`             | Duplicate               |
| `Fraudulent`            | Fraudulent              |
| `OrderChange`           | Order change            |
| `ProductUnsatisfactory` | Product unsatisfactory  |
